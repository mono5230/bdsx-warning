#### 운영자 전용
```
- /경고 <경고 줄 닉네임> <경고 횟수> <사유>
```
```
- /경고차감 <경고 차감할 닉네임> <차감 할 경고 수>
```
```
- /경고확인 <경고를 확인 할 닉네임>
```
```
- 플러그인 파일 안에 있는 Warningplayer 라는 파일은 경고 수 등을 저장 하기 위에 쓰입니다.
```
